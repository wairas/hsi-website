# WAILight: Spectrally Tunable Illumination System

The **WAILight** is a custom-built, spectrally tunable luminaire designed to bridge the gap between computational imaging and physical illumination. Unlike standard lighting systems, WAILight can algorithmically optimize its output to maximize contrast for machine vision applications.

<img src = '/wailight/2025 WAILightOnFrame.jpeg' alt = 'image11' width = '700'/>

## System Overview

The core of the WAILight system is a programmable array of monochromatic LED channels covering the visible spectrum. By precisely modulating the brightness of each channel, the system can:

- Synthesize standard illuminants (e.g., CIE D50, D65) on demand.
- Generate "hyper-contrast" spectra designed to reveal specific features (e.g. in forensic samples).
- Operate as a "Physical AI" agent, adjusting lighting in real-time based on feedback from imaging sensors.

## Applications
### Forensic Imaging

<img src = '/wailight/WhatsApp Image 2025-12-17 at 18.31.41.jpeg' alt = 'image12' width = '700'/>

One of the primary application of WAILight is in **Forensic Cloth Contrast Enhancement**. By tuning the illumination spectrum to exploit the optical properties of specific biological stains or fabric dyes, the system can render invisible evidence visible without chemical treatment.

### Computational Color Science

<img src = '/wailight/WhatsApp Image 2025-12-17 at 18.31.41_1.jpeg' alt = 'image12' width = '700'/>

WAILight serves as a testbed to physically validate theoretical optimal lighting conditions for object detection and segmentation.

### Camera Calibration

<img src = '/wailight/WhatsApp Image 2025-12-17 at 18.31.41_2.jpeg' alt = 'image12' width = '700'/>

WAILight is used for high-precision camera calibration, offering superior performance compared to traditional monochromator and integrating sphere setups. Its high-intensity output eliminates the noise sensitivity issues common in those low-throughput methods, ensuring robust and accurate sensor characterization. This allows us to ground-truth our imaging pipelines and ensure accurate color reproduction across different sensors.

### Case Study: Far-Red Photobiology Panels

<img src = '/wailight/WhatsApp Image 2025-12-17 at 18.31.41_3.jpeg' alt = 'image12' width = '700'/>

We recently delivered a set of specialized illumination rigs for the School of Science to support research into cyanobacteria photosynthesis. The research group required a specific "shaded" spectral environment that commercial grow lights could not provide. We engineered panels capable of high-intensity output specifically in the Far-Red (700nm+) region to simulate these niche conditions.
